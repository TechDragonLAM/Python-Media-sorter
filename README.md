# Python-Media-sorter
Simple desktop app to organize pictures and videos by year and month.

## Features
- Sorts images and videos automatically
- Uses a clean GUI
- Progress bar during sorting
- No internet access required
- No files deleted

## How to Use
1. Download the `.exe` from the Releases section
2. Run the app
3. Select your source folder
4. Select destination folders for pictures and videos
5. Click **Sort Media**

## Notes
- The app only moves files you select
- Windows may show a security warning because the app is unsigned

## Antivirus Notice
This application is unsigned and may trigger false positives in some antivirus engines.
It does not collect data, access the internet, or install persistence mechanisms.
Source code is provided for transparency.
